01
What Is a Variable?
Learning how Python stores and names pieces of information in your computer's memory.
01 · Variables
What Is a Variable?
Imagine you are solving a maths problem on paper. You write: "Let x = 5." You have given the name x to the value 5. Later in the problem, whenever you write x, you mean 5.
Python works in exactly the same way. You give a name to a piece of information, and Python remembers it for you.
📖 Definition
A variable is a named storage location in the computer's memory. You choose the name, and Python stores the value under that name.
🗃️ Analogy: A Labeled Box
Think of a variable as a labeled box:
20
↑ value inside
age
↑ label (name)
You can change the contents any time. The label stays the same.
01 · Variables
Storing a Value — The Assignment Operator
To store a value in a variable, we use the assignment operator: the equals sign =.
⚠️ Important Distinction
In programming, = does NOT mean "is equal to." It means "store the value on the right into the variable on the left."
Example 1: Store and Print
Python
age = 20
print(age)
Output
20
Line 1: Python creates age and stores 20. Line 2: Retrieves the value and prints it.
Example 2: Changing a Value
Python
score = 10
print(score)

score = 95
print(score)
Output
10
95
The variable is reassigned — old value is replaced.

02
Basic Data Types
Understanding int, float, and str — the three fundamental kinds of data in Python.
02 · Data Types
What Are Data Types?
Not all information is the same. The number 42 is different from the text "hello". Python uses data types to distinguish between different kinds of information.
📖 Definition
A data type tells Python what kind of value a variable holds and what operations can be performed on it.
The three fundamental data types we study today:
int
Integer
Whole numbers. No decimal point.
Examples: 5, 100, -3, 0
float
Float
Decimal numbers.
Examples: 3.14, 0.5, -1.7
str
String
Text, characters, symbols in quotes.
Examples: "hello", "Ahmed"
02 · Data Types
Integers — int
📖 Definition
An integer is a whole number — no decimal point. It can be positive, negative, or zero.
Examples: 0  1  -10  500  2025
Python
year = 2025
temperature = -3
count = 0

print(year)
print(temperature)
print(count)
Output
2025
-3
0
✅ Note
Each variable stores a whole number. There are no decimal points. Python recognises these as integers automatically.
02 · Data Types
Floats — float
📖 Definition
A float (floating-point number) includes a decimal point. Use floats when precision beyond whole numbers is needed.
Examples: 3.14  0.5  -2.7  100.0
Python
price = 9.99
height = 1.75
pi = 3.14159

print(price)
print(height)
print(pi)
Output
9.99
1.75
3.14159
⚠️ Important
Even 100.0 is a float because it contains a decimal point, even though its value is a whole number.
02 · Data Types
Strings — str
📖 Definition
A string is a sequence of characters — letters, digits, spaces, or symbols — enclosed in quotation marks. Used to represent text.
You can use single quotes '...' or double quotes "...".
Python
first_name = "Sara"
greeting = "Hello, welcome to Python!"
course_code = "CS101"

print(first_name)
print(greeting)
print(course_code)
Output
Sara
Hello, welcome to Python!
CS101
⚠️ Watch Out
Even "101" is a string (text), not a number — because it is enclosed in quotes!
02 · Data Types
Checking the Data Type with type()
Python provides a built-in function called type() that tells you the data type of any value or variable.
Python
age = 20
gpa = 3.75
name = "Ali"

print(type(age))
print(type(gpa))
print(type(name))
Output
<class 'int'>
<class 'float'>
<class 'str'>
int
age = 20
A whole number
float
gpa = 3.75
A decimal number
str
name = "Ali"
Text in quotation marks
You will use type() frequently to check whether a variable holds the kind of value you expect.
03
Getting Input from the User
Using input() to make programs interactive and respond to the user at runtime.
03 · User Input
How input() Works
So far, all values in our programs have been written directly in the code. Most programs need to interact with the user — asking for their name, age, or other information.
The 3 Steps of input()
  1. Displays a prompt message on the screen
  2. Pauses and waits for the user to type something
  3. When the user presses Enter, Python captures what was typed and stores it in a variable
Python
name = input("What is your name? ")
print("Hello,", name)
Example Interaction
What is your name? Sara
Hello, Sara
💡 Tip
Add a space before the closing quote in the prompt — "name? " — to create a small gap between the prompt and the user's typed text.
03 · User Input
⚠️ Critical Rule: input() Always Returns a String
⚠️ Most Important Rule
input() always returns a string (str), regardless of what the user types.
Even if the user types the number 25, Python stores it as the text "25", not the number 25.
Demonstration
Python
age = input("How old are you? ")
print(type(age))
Interaction
How old are you? 25
<class 'str'>
Even though the user typed 25, type(age) shows str.
What Happens If You Try Arithmetic
age = input("Your age? ")
print(age + 1)
# TypeError: can only concatenate
# str (not "int") to str
You cannot perform arithmetic on text. This is why type conversion is needed!
04
Type Conversion
Converting strings from input() into numbers using int() and float().
04 · Type Conversion
Converting Input to a Number
To use a number entered by the user in calculations, you must convert it from a string to a number. This is called type conversion (or type casting).
int()
Convert to Whole Number
Use when the user should enter a whole number.
int("25") → 25
float()
Convert to Decimal Number
Use when the user may enter a decimal.
float("3.5") → 3.5
Example — Two-step conversion
Python
age_text = input("How old are you? ")
age = int(age_text)
print("Next year you will be", age + 1)
Output
How old are you? 20
Next year you will be 21
Example — One-step (common style)
Python
age = int(input("How old are you? "))
print("Next year you will be", age + 1)
Python runs input() first, then immediately passes the result to int(). Equivalent to the two-step version.
04 · Type Conversion
Using float() for Decimal Input
Use float() when the user may enter a decimal number.
Python
height = float(input("Enter your height in metres: "))
print("Your height is", height, "metres.")
Interaction
Enter your height in metres: 1.75
Your height is 1.75 metres.
How to choose: int() vs float()?
int()
Age, count, year, number of students
float()
Height, weight, price, GPA, temperature
⚠️ Cannot Use int() on Decimals
int("3.14") causes an error! Use float("3.14") instead.
05
Combining Variables in Output
Two clean ways to include variable values inside printed messages.
05 · Output
Two Ways to Include Variables in Output
Method 1 — Commas in print()
Simple: separate values with commas. Python adds a space between each.
Python
name = "Lena"
age = 22
print("Name:", name, "Age:", age)
Output
Name: Lena Age: 22
Method 2 — f-strings (Recommended)
Place f before the quote, then use {variable} inside.
Python
name = "Lena"
age = 22
print(f"My name is {name} and I am {age} years old.")
Output
My name is Lena and I am 22 years old.
✅ Why f-strings?
Cleaner, more readable, widely used in Python. Python replaces {name} with the actual value of name.

Common Mistakes
Five frequent errors beginners make — and how to fix them.
06 · Common Mistakes
Mistakes 1 & 2
❌ Mistake 1: = vs ==
Confusing assignment = with comparison ==
Python
# This STORES 10 in x.
# It does NOT ask "is x equal to 10?"
x = 10
= is assignment (stores a value).
== is comparison (asks if values are equal). We study == later.
❌ Mistake 2: Using a Variable Before Assigning It
Python reads code top-to-bottom. Use before define = error!
Error
print(score)  # Error!
score = 85    # Too late!
✅ Fix
Always assign a value to a variable before using it.
06 · Common Mistakes
Mistakes 3, 4 & 5
❌ Mistake 3: Arithmetic on a String
input() returns text. You cannot add a number to text!
Wrong
age = input("Enter age: ")
print(age + 1)  # Error!
Fixed
age = int(input("Enter age: "))
print(age + 1)  # Works!
❌ Mistake 4: Missing Quotes Around Strings
Wrong
name = Sara  # Error!
Fixed
name = "Sara"
❌ Mistake 5: int() on a Decimal String
Wrong
int("3.14")  # Error!
Fixed
float("3.14") # ✓
