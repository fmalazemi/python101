01 · Introduction
What Is a Conditional Statement?
📖 Definition
A conditional statement is an instruction that tells the program:
"Do this only if a certain condition is true."
In Python, the three main keywords for making decisions are:
if elif else
These keywords let you tell Python: "Check this condition, and decide what to do next."
🚦 Analogy: Traffic Light
When the light is red, you stop. When it is green, you go. The driver decides what to do based on the condition (light colour).

A conditional statement works exactly the same way.
→ Stop
→ Slow
→ Go

Comparing Values in Python

⚠️ Critical Distinction
== checks if two values are equal (comparison). = assigns a value to a variable (assignment). These are completely different! Mixing them up is one of the most common beginner mistakes.

02
The if Statement
Run code only when a condition is True
02 · The if Statement
The if Statement
The if statement tells Python: "Run this block of code only if the condition is true." If the condition is false, Python skips the block entirely.
if condition:
# code runs only if True
print("something")
⚠️ Two Mandatory Rules
1. A colon : after the condition is required.
2. The code inside must be indented 4 spaces.
🎫 Analogy
Imagine a door: "Enter only if you have a ticket." No ticket? Walk past. The if statement works exactly like this sign.
Example — Check Positive Number
Python
number = 8

if number > 0:
    print("The number is positive.")
Output
The number is positive.
✅ What Happens
8 > 0 is True → the indented block runs.
Change to number = -3 and nothing prints — the block is skipped.

03 · if–else
The if–else Statement
The else provides an alternative. It says: "If the condition is true, do this. Otherwise, do that." Only one block ever runs — never both.
if condition:
# runs if True
else:
# runs if False
☂️ Analogy
You check the weather before leaving. If it is raining, take an umbrella. Otherwise, leave without one. Exactly two possibilities — you choose one based on the condition.

03 · if–else
Examples — Pass/Fail and Even/Odd
Pass or Fail
Python
score = 55

if score >= 50:
    print("Result: Pass")
else:
    print("Result: Fail")
Output
Result: Pass
55 >= 50 is True → first block runs. Change score to 40 and the else block runs instead.
Even or Odd
Python
number = 7

if number % 2 == 0:
    print("The number is even.")
else:
    print("The number is odd.")
Output
The number is odd.
💡 The % Operator
% gives the remainder. 7 % 2 = 1, not 0 → the number is odd.

The if–elif–else Statement
Handle three or more possibilities in sequence
04 · if–elif–else
Checking Multiple Conditions with elif
Sometimes there are more than two possibilities. The elif keyword (short for else if) lets you check multiple conditions in sequence.
Python checks from top to bottom and runs only the first block whose condition is True. If none match, the else block runs.
if condition_1:
# if condition_1 is True
elif condition_2:
# if condition_2 is True
elif condition_3:
# if condition_3 is True
else:
# if none above are True
🚦 Traffic Light — 3 Conditions
If green → go.
Else if yellow → slow down.
Else if red → stop.

You check each in order. The first match wins.
✅ Key Rule
Python stops checking as soon as it finds the first True condition. The remaining branches are never evaluated.
📝 Note
The final else is optional, but good practice — it catches anything that did not match.
04 · if–elif–else
Example — Grade Classification
Python
score = 72

if score >= 90:
    print("Grade: A")
elif score >= 75:
    print("Grade: B")
elif score >= 60:
    print("Grade: C")
elif score >= 50:
    print("Grade: D")
else:
    print("Grade: F")
Output
Grade: C
How Python Traces This (score = 72)
72 ≥ 90 → FalseMove to next
72 ≥ 75 → FalseMove to next
72 ≥ 60 → True✓ Run this block
72 ≥ 50Skipped
elseSkipped
Python does not check remaining conditions once it finds the first True result.

Indentation

05 · Indentation
Indentation — A Critical Rule in Python
📖 What Is Indentation?
Indentation means adding spaces at the beginning of a line to show it belongs to a block. In Python, indentation is not optional — it is a rule of the language. Use 4 spaces per level.
❌ Wrong — Error
IndentationError
if score >= 50:
print("Pass")
✅ Correct
Python
if score >= 50:
    print("Pass")
Multiple Lines in One Block
You can have many lines inside one block — all must be indented equally.
Python
temperature = 38

if temperature > 37.5:
    print("You have a fever.")
    print("Please rest and drink water.")
    print("Consider seeing a doctor.")
else:
    print("Your temperature is normal.")
Output
You have a fever.
Please rest and drink water.
Consider seeing a doctor.
All three print statements belong to the if block — same indentation level.

06 · User Input
Combining input() with Conditionals
In real programs, conditions depend on user input. This example combines input(), type conversion, and a conditional statement.
Python
age = int(input("Enter your age: "))

if age >= 18:
    print("You are eligible to vote.")
else:
    print("You are not yet eligible to vote.")
Sample: user enters 20
Enter your age: 20
You are eligible to vote.
Sample: user enters 15
Enter your age: 15
You are not yet eligible to vote.
⚠️ Always Convert input()
input() always returns a string.

If you forget int(), comparing the string to the number 18 will produce a TypeError.
✅ The Pattern to Remember
variable = int(input("prompt"))

Always wrap input() with int() or float() before comparing to a number.

Common Mistakes
Five errors beginners make — and how to fix each one
07 · Common Mistakes
Mistakes 1 & 2
❌ Mistake 1: Using = instead of ==
Inside if conditions, you almost always need ==, not =.
❌ Wrong
SyntaxError
if score = 80:
    print("Correct")
✅ Fixed
Python
if score == 80:
    print("Correct")
❌ Mistake 2: Missing the Colon
Every if, elif, and else line must end with a colon :.
❌ Wrong
SyntaxError
if score >= 50
    print("Pass")
✅ Fixed
Python
if score >= 50:
    print("Pass")
07 · Common Mistakes
Mistakes 3, 4 & 5
❌ Mistake 3: Missing Indentation
❌
IndentationError
if score >= 50:
print("Pass")
✅
Fixed
if score >= 50:
    print("Pass")
❌ Mistake 5: else Without if
else cannot stand alone. It must always follow an if or elif.
❌
SyntaxError
else:
    print("None")
✅
Fixed
if score >= 50:
    print("Pass")
else:
    print("Fail")
❌ Mistake 4: Comparing a String to a Number
input() always returns a string. You must convert it with int() before comparing to a number.
❌ Wrong — TypeError
TypeError
age = input("Enter age: ")
if age >= 18:
    print("Adult")
✅ Fixed
Python
age = int(input("Enter age: "))
if age >= 18:
    print("Adult")
Summary
Lecture 4 — Key Concepts
  • A comparison operator compares two values and returns True or False
  • An if statement runs a block only when a condition is True
  • An if–else statement chooses between exactly two blocks
  • An if–elif–else chain handles three or more possibilities, checking in order
  • Indentation is mandatory — the body of every block must be indented 4 spaces
  • Use == for comparison, = for assignment — never mix them up
  • Always convert user input with int() or float() before numeric comparisons
StructureUse When
if onlyDo something only if condition is true
if–elseChoose between exactly two actions
if–elif–elseChoose between 3 or more actions
