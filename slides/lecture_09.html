01 · What Is a List
What Is a List?
📖 Definition
A list is an ordered collection of values stored in a single variable. Each value is called an element (or item).
Lists in Python are:
  • Ordered — elements keep their position
  • Mutable — you can change, add, or remove elements
  • Zero-indexed — the first element is at index 0
  • Mixed — can hold numbers, strings, or both
🗄️ Analogy: A Numbered Shelf
Imagine a shelf with numbered slots. Each slot holds one object. The shelf has one name, but each slot is identified by a number.

03 · Updating
Updating Elements — Lists Are Mutable
📖 Mutability
Unlike strings, lists are mutable — you can change any element directly by assigning a new value to a specific index.
list_name[index] = new_value
Python
marks = [78, 85, 90, 62, 74]
print("Before:", marks)

marks[3] = 70   # Change index 3 (was 62)
print("After:", marks)
Output
Before: [78, 85, 90, 62, 74]
After:  [78, 85, 90, 70, 74]
The element at index 3 changed from 62 → 70. All other elements stay the same.

03 · Methods
remove() and len()
📖 remove(value)
Removes the first occurrence of the specified value. Elements shift to fill the gap.
Python
shopping = ["milk", "bread", "eggs", "butter"]
shopping.remove("bread")
print(shopping)
Output
['milk', 'eggs', 'butter']
⚠️ ValueError if Not Found
If the value does not exist, Python raises a ValueError. Always check first, or handle the error.
📖 len(list)
Returns the total number of elements in the list. It is a function, not a method.
Python
marks = [78, 85, 90, 62, 74]
total_students = len(marks)
print("Students:", total_students)  # 5

# Combined with indexing:
# Last valid index = len(marks) - 1
print(marks[len(marks) - 1])  # 74
Output
Students: 5
74
💡 Last Index Formula
Last valid index = len(list) - 1    or simply use list[-1]

💡 Key Points
List indices start at 0. The last valid index is len(list) - 1. append() adds to the end. remove() removes by value. Accessing a non-existent index causes an IndexError.

04 · Looping
Looping Through a List — Direct Iteration
📖 The Pattern
Use a for loop to visit each element one by one. The loop variable takes the value of each element in turn.
Python
students = ["Ali", "Sara", "Omar", "Lena"]

for student in students:
    print(student)
Output
Ali
Sara
Omar
Lena
The variable student holds each name in turn. You can name it anything — choose something descriptive.
What happens on each iteration
Iter 1→student = "Ali"→print "Ali"
Iter 2→student = "Sara"→print "Sara"
Iter 3→student = "Omar"→print "Omar"
Iter 4→student = "Lena"→print "Lena"
✅ Two Styles of Loop
for item in list: — gives you the value directly

for i in range(len(list)): — gives you the index (use when you need the position)
04 · Looping
Summing a List and Calculating the Average
Sum All Elements
Python
marks = [78, 85, 90, 62, 74]
total = 0

for mark in marks:
    total = total + mark

print("Total marks:", total)
Output
Total marks: 389
total starts at 0. Each mark is added to it. After the loop, total = 78+85+90+62+74 = 389.
Calculate the Average
Python
marks = [78, 85, 90, 62, 74]
total = 0

for mark in marks:
    total = total + mark

average = total / len(marks)
print("Average mark:", average)
Output
Average mark: 77.8
💡 Using len() for the Average
Dividing by len(marks) is better than hard-coding 5 — it works correctly no matter how many elements are in the list.

05 · Practical Examples
Example — Printing a Numbered List
📋 Task
Print each shopping item with a number (starting from 1), using range(len()).
Python
shopping = ["milk", "bread", "eggs", "butter"]

for i in range(len(shopping)):
    print(i + 1, "-", shopping[i])
Output
1 - milk
2 - bread
3 - eggs
4 - butter
range(len(shopping)) generates 0, 1, 2, 3. On each iteration, i is the index. shopping[i] retrieves the element. i + 1 makes the display start at 1 instead of 0.
for item in list
Use when you only need the value
for i in range(len())
Use when you need the index too (for numbering, comparing, etc.)

05 · Practical Examples
Example — Finding the Highest Mark
Python
marks = [78, 85, 90, 62, 74]
highest = marks[0]   # Assume first is highest

for mark in marks:
    if mark > highest:
        highest = mark

print("Highest mark:", highest)
Output
Highest mark: 90
How it works — step by step
Init: highest = 78 (first element)
85 > 78 → True → highest = 85
90 > 85 → True → highest = 90
62 > 90 → False → no change
74 > 90 → False → no change
Result: highest = 90 ✓
🎯 Try It
Modify to find the lowest mark. Change > to < and initialise with marks[0].

Five errors beginners make — and how to fix each one
06 · Common Mistakes
Mistakes 1 & 2
❌ Mistake 1: Index Out of Range
Valid indices run from 0 to len(list) - 1. Accessing beyond causes an IndexError.
❌ IndexError
IndexError
marks = [78, 85, 90]
print(marks[3])
# 3 elements: valid = 0,1,2
✅ Fixed
Fixed
marks = [78, 85, 90]
print(marks[2])  # last = 2
print(marks[-1]) # or use -1
❌ Mistake 2: Forgetting Index Starts at 0
Many beginners expect the first element to be at index 1. It is at index 0.
❌ Wrong Element
Gets "Sara" not "Ali"
students = ["Ali", "Sara", "Omar"]
print(students[1])
# Expected "Ali", got "Sara"!
✅ Fixed
Fixed
students = ["Ali", "Sara", "Omar"]
print(students[0])  # "Ali" at 0
06 · Common Mistakes
Mistakes 3, 4 & 5
❌ Mistake 3: remove() with Missing Value
If the value is not in the list, remove() raises a ValueError. Check first!
❌ ValueError
ValueError
shopping = ["milk", "bread"]
shopping.remove("butter")
✅ Check First
Fixed
if "butter" in shopping:
    shopping.remove("butter")
❌ Mistake 5: len() as the Last Index
len() returns the count. The last valid index is len - 1, not len.
❌ IndexError
IndexError
marks = [78, 85, 90]
print(marks[len(marks)])
# 3 elements → index 3 invalid!
✅ Fixed
Fixed
print(marks[len(marks) - 1])
# or more simply:
print(marks[-1])
❌ Mistake 4: Modifying a List While Looping Over It
Changing the size of a list (adding/removing elements) while looping through it can cause items to be skipped or unexpected errors.
❌ Problematic
Unexpected behaviour
for number in numbers:
    if number == 3:
        numbers.remove(number)  # Avoid!
✅ Better Approach
Fixed
# Loop over a copy, or collect
# items to remove, then process
# them after the loop ends.
Summary
Lecture 9 — Key Concepts
  • A list is an ordered, mutable collection stored in one variable
  • Create with square brackets: [v1, v2, v3] or [] for empty
  • Access with list[index] — indices start at 0
  • Negative indices count from the end: list[-1] = last element
  • list[i] = value updates an element in-place (mutable!)
  • append() adds to end · remove() removes by value · len() counts
  • Last valid index is always len(list) - 1
  • Use for item in list: to process all elements
Quick Reference
marks = [78, 85, 90, 62, 74]

marks[0]         # 78 (first)
marks[-1]        # 74 (last)
marks[2] = 95    # update index 2

marks.append(88) # add to end
marks.remove(62) # remove value

len(marks)       # count elements

# Loop — direct iteration
for m in marks:
    print(m)

# Loop — with index
for i in range(len(marks)):
    print(i + 1, marks[i])
